FEATURES SUMMARY

Each game is represented by a circle on the graph, the top half is based on the home team's information
and the bottom half is based on the away team.

The X axis is the rounds and the Y axis is the score
You can change the Y axis to be based on the total score, away team's score or the home team's score
by clicking the labels on the left.
The X axis can be focused on the regular season, the finals or both by using the season/finals filter


The top bar shows the filters and colours and a reset button

You can change the fill colour and stroke colour for each game based on 
the year, team, team's country and whether they were home or away.

A key for the current colour palette is on the bottom left
If you select a particular game a summary appears on the right
